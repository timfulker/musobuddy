Client Portal Mobile View Test

Testing if the navigation menu is hidden on the client portal page in mobile view.

Mobile View (iPhone 12 Pro - 390x844)

Instructions for Testing:

  1. Open the developer tools (F12)
  2. Toggle device emulation (Ctrl+Shift+M or click the mobile icon)
  3. Select a mobile device (e.g., iPhone 12 Pro)
  4. Navigate to a client portal URL with a token parameter
  5. Check that NO navigation menu appears at the bottom of the screen
  6. Verify that the save button (if visible) stays at the bottom and is accessible
